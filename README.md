
# php-coursework1-bookstore

### Short description
This mini-bookstore app for the coursework displays a book list, applies discounts, and allows users to add new books.
### How to run:
- Location  
put this file into `/www/phpnov1`

### features
#### Adding a book
Access to this link `http://localhost/phpnov1/php-coursework1-bookstore/`
and enter the information of the book.
The application then displays the added book and the total price after discounts. Simultaneously, it adds a log containing the new book's information.


#### Display Booklist
Access to this link `http://localhost/phpnov1/php-coursework1-bookstore/index.php/booklist`
It shows Booklist of this app.

#### Discount book
If the book's genre is `Science Fiction`. This app discounts 10 percent of the price.
